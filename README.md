# How to compile TypeScript examples locally

If you want to build a TypeScript example from the TypeScript Section you need to have to install "typescript" module globally:

> npm i typescript -g

Go to an example folder and type:

> npm install

Now you can build the example:

> npm run debug

Open index.html using your local server tool. I use [http-server](https://www.npmjs.com/package/http-server)
